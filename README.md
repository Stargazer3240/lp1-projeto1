# IMDJ
IMDJ is a CLI music library manager made for the first Project of the course LP1, from BTI-UFRN. 

## Instalation

### Requirements
- C++ compiler on your PATH
- CMake
- Git (optional)

### Compilation
``` 
git clone https://github.com/Stargazer3240/lp1-projeto1.git
cd ./lp1-lista1
cmake -S . -B build/
cmake --build build/
./bin/program
```
